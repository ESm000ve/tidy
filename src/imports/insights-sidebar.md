Add an optional “Insights” sidebar that summarizes what the system detects before and after a run.

This should feel informational — not promotional.

Purpose

Move the app beyond simple sorting by surfacing patterns in the file system.

Examples:

47 screenshots detected

23 duplicate images

18 PDFs appear to be invoices

9 audio files missing metadata

This reframes the app as file intelligence, not just file movement.

Placement

Add a collapsible right sidebar:

Title:
Insights

Icon:
Small sparkle or graph icon

Collapsed by default.

Layout Structure

Group insights into sections:

File Types

Screenshots: 47

Photos: 128

Screen recordings: 12

Duplicates

23 duplicate groups

Largest duplicate: 48 MB

Documents

18 likely invoices

6 contracts detected

Audio

9 files missing artist metadata

Each insight should include a small action link:

[ View in Preview ]
[ Review Duplicates ]

Behavior

Insights update:

After scan

After run

When toggles change

They should never auto-trigger changes.

Visual Tone

Must feel:

Subtle

Utility-focused

macOS-like

Clean and informational

Avoid:

Charts

Big colorful cards

Gamification

AI hype visuals

Think:
Activity Monitor meets Finder.

Design Goal

Users should feel:

“The app understands my files.”

Not:

“This is a dashboard.”
