# Tidy - File Organizer 

Tidy is a modern, beautifully designed file organizing application for macOS, packed with powerful features to help you keep your directories clean. Designed with standard macOS interface principles but updated with a dynamic UI, Tidy brings elegance and simplicity to what’s usually a tedious task.

## Features

- **Dynamic Theme** - Supports Light, Dark, and System appearance.
- **Rule Engine** - Visually build rules to filter files by type, extension, string match, and date ranges. 
- **Natural Language Parsing** - Simply type "Move all images from last week into my Archives" and let the AI generate a rule for you.
- **Preview Operations** - Inspect exactly what will be moved and where *before* running any destructive operations. Handles file collisions/duplicates gracefully.
- **Intelligent Insights** - Detects common patterns in your folders to suggest rules that might save you time.
- **Automated Scheduling** - Run your cleanups automatically using the visually stunning time picker.

## Getting Started

1. Clone or download the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the Electron development server.
4. Run `npm run build` to compile a macOS application DMG suitable for shipping!
